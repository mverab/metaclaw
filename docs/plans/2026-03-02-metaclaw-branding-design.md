# MetaClaw Branding Design

## Objective
Create non-generic, product-faithful branding assets for the public repository that reflect MetaClaw as a MetaLayer product focused on founders.

## Audience
- Primary: founders building AI operations.
- Excluded from primary narrative: agency-first and community-ops framing.

## Product Value Proposition
MetaClaw is the meta-layer that turns intent into an executable multi-agent operating setup.

### Messaging Pillars
1. Speed to operating system: move from idea to an installable setup.
2. Strategic control: clear architecture and role separation.
3. Execution reliability: reproducible, structured outputs.

## Visual Direction
- Direction: Precision Ops.
- Symbolism: abstract MetaLayer geometry; no literal claw icon.
- Tone: technical, premium, founder-serious, minimal.
- Text in art: minimal only (`MetaClaw` + `Setup Architect`).

## Visual System
- Base: `#0B1220`, `#111827`
- Accents: `#22D3EE`, `#A3E635`
- Text: `#E5E7EB`
- Composition: strong negative space, clean geometric layering, high contrast.

## Asset Set
1. Logo (square and horizontal-safe variant)
2. Repo banner (hero image for README)
3. Social preview card (Open Graph 1200x630)

## Selection Criteria
1. Legibility at small size (logo first).
2. Cohesion across all assets.
3. Founder-grade credibility.
4. Distinctiveness without over-styling.

## Approved Constraints
- Keep scope small and practical.
- Avoid overengineering in process and deliverables.
- Use AI image generation outputs, not hand-drawn placeholder vectors.
